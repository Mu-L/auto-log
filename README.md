- 极简 api，一行代码搞定一切

# 快速开始

## maven 引入

```xml
<dependency>
    <group>com.github.houbb</group>
    <artifact>auto-log</artifact>
    <version>0.0.1</version>
</dependency>
```

## 入门例子

- 示例代码

```java
List<Long> resultList = LuckyMoneyHelper.luckyMoney(1000, 5);
```

- 结果如下

```xml
[269, 57, 42, 190, 89, 354]
```
